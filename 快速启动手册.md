## 📋 您的启动步骤

### **1. 启动PC管理端 (测试用)**
```bash
cd admin-console
npm install
npm run dev
# 访问: http://localhost:8080
```

### **2. 启动后端服务**
```bash
cd backend
npm run dev
# 服务地址: http://localhost:3000
# 健康检查: http://localhost:3000/health
```

### **3. 创建小程序项目 (您的任务)**
1. **打开 HBuilderX**
2. **文件 → 导入 → 从本地目录导入**
3. **选择 miniprogram/ 目录**
4. **选择 uni-app 项目模板**
5. **配置 manifest.json 和 pages.json**
6. **运行 → 运行到小程序模拟器 → 微信开发者工具**

### **4. 创建移动管理项目 (您的任务)**
1. **同样步骤导入 mobile-admin/ 目录**
2. **创建管理端项目**
3. **运行到微信开发者工具**

## 🔧 基础配置

### **小程序API配置**
在小程序项目中配置后端接口：
```javascript
// utils/api.js
const API_BASE_URL = 'http://localhost:3000/api'

// 示例请求
export const request = (options) => {
  return uni.request({
    url: API_BASE_URL + options.url,
    method: options.method || 'GET',
    data: options.data,
    header: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${uni.getStorageSync('token')}`
    }
  })
}
```

### **Docker服务问题**

